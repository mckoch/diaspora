Automatic

  1. Unpack the archive.
  2. Double click the included installer.
  3. Fit the paths during installation, if necessary (see manual install).

Manual

  1. Copy dsp_pack.dat to the user plugins directory, usually %USERDIR%\SimCity 4\plugins.
  2. Create a blank directory 'Diaspora' under %SC4PATH%\Apps.
  3. Copy patch.dat and data.dat to %SC4PATH%\Apps\Diaspora.
  4. Make sure %SC4PATH%\Apps\Diaspora\ and the files below are writable.

Automatic and manual: testing the installation

  1. Start sc4 and load a blank city (just to make sure...)
  2. As soon as mayor mode is entered,
  3. check Apps\data.dat which should have been updated and grown right now. It is a blank text file, readable by any editor.
  4. Open Apps\patch.dat in a text editor and start having fun.

Files locations

Plugin is installed according to manual install, for documentation %USERDIR%\SimCity 4\Diaspora would be a common place. For quick technical information and how to: dsp_readme.html, for a lot of material to experiment with: configMarkdown.html. dsp_params.html will be very useful, maybe inspiring...
